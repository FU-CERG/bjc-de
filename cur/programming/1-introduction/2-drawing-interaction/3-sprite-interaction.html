Sprite Interaktionen

Ändere jetzt das Skript, sodass die Sprites sich nicht fortlaufend bewegen, sondern nur wiederhole bis sie sich berühren.

  1. Ändert den Code von Sprite(2) so, dass er nur so lange auf Sprite zeigt, bis er Sprite berührt. Die folgenden Anweisungen und Animationen zeigen euch, wie das geht.
    1. Wählt Sprite(2) durch Klicken seines Symbold unterhalb der Bühne aus.
    2. Ersetze den fortlaufend Block durch einen wiederhole bis Block.
    3. Ziehe in das sechseckige weiße Feld im wiederhole bis Block. Stellen sie ihn so ein, dass er sich wiederholt bis er Sprite berührt.
  2. Mache das gleiche für Sprite. Ändere seinen Code so, dass er nur so lange der Maus folgt bis Sprite(2) berührt wird.
  3. Testet euer Programm, um zu sehen ob es das tut was ihr wolltet:
    • Wenn du auf klickst, sollte Sprite (grau) deiner Maus folgen und Sprite(2) sollte sich drehen um auf Sprite zu zeigen.
    • Wenn sich beide Sprites so nah kommen, dass sie sich berühren, sollten beide ihre Bewegungen stoppen.
  4. Bis jetzt zeigt Sprite(2) in Richtung Sprite, aber bewegt sich nicht in dessen Richtung. Füge eine Zeile Code zum Skript von Sprite(2), sodass es Sprite verfolgt. Hier ist eine Idee:
    Fülle das Eingabefeld im gehe Block mit einer kleinen Zahl, sodass Sprite(2) Sprite nicht zu schnell fängt.
  5. Testet euer Programm ein Paar Mal.

  6. Bis jetzt stoppen beide Sprites nur wenn sie sich treffen. Lasst die beiden Sprites ein Gespräch führen, wenn sie sich treffen. Entwickelt eure eigenes Gespräch. Ihr könnt jede Sprache nehmen, die ihr eingeben könnt.

    Sprite(2) macht die Dinge selbst. Er sagt Sprite auch, was zu tun ist und wann es zu tun ist. Dieser Code überträgt Sprite(2) die Verantwortung für Sprite.

    In der aktuellen Version von snap, ist der tell Block nicht automatisch installiert. Klicke auf das Datei Menü oben im snap Fenster und wähle "Module...". Der sprich Block ist im Unterpunkt Sprachausgabe enthalten. Klicke auf importieren und die Blöcke werden am unteren Ende in der Palette erscheinen. Ihr werdet die Importfunktion bei jedem neuen snap projekt erneut benötigend, bei denen ihr sprich Blöcke verwendet.

Ihr könnt das Projekt auf verschiedenste Art Weise personalisieren...

  1. Du kannst den Sprites Kostüme geben. Ihr könnt sogar ein Foto oder Bild aus dem Internet durch Ziehen des Fotos in den Kostümbereich des Sprites. Falls das ausgesuchte Bild zu groß ist, setze die Größe auf etwas kleineres als 100% durch benutzen von .
  2. Ihr könnt ein Dialog über Schule, über Zu Hause oder über die Arbeit entwerfen.
  3. Ihr könnt einen dritten Sprite hinzufügen und diesen ebenfalls mit den vorhandenen Sprites interagieren lassen.
  4. Ihr könnt sogar den Hintergund eurer Bühne ändern.
