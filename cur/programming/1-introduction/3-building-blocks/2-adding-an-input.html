Übergebt euren Blöcken eine Input-Variable

Ihr habt bereits viele Blöcke gesehen und benutzt, die einen Input benötigen:






Euer zeichne Quadrat Block zeichnet Quadrate in nur einer festgelegten Größe—Die Größe wird durch das Skript festgelegt. Ihr könnt euren Blöck verbessern, indem Quadrate in jeder Größe zeichnen kann. Das wird durch einen Input realisiert.

  1. Bearbeitet euren zeichne Quadrate Block, sodass er einen Input benötigt.
    • Um einen Block zu bearbeiten, macht ihr einen Rechtsklick (oder control-Klick) auf den Block und wählt "bearbeiten" aus.
    • Um einen Input hizuzufügen, klickt ihr auf das rechteste Pluszeichen in eurem Block, tippt ein Input-Name (Größe) ein, und klickt auf "OK."
    • Damit ihr die eingegebenen Daten verwenden könnt, zieht ihr den Input-Name (Größe) aus dem Kopfblock heraus und platziert ihn in eurem Skript.
    Benutzt die Animation, um eine Idee von diesem Vorgehen zu beommen. Ihr könnt die Details etwas unterhalb im gelben Kasten lesen, um ein besseres Verständnis dafür zu bekommen.
  • Falls ihr die + Zeichen nicht seht, müsst ihr das Häckchen bei "Ausführlicher Input Dialog" im Einstellungsmenü setzen.
    Die + Zeichen: In dem Block Editor schwebt ihr im Kopfblock mit der Maus über die Wörter und plus Zeichen (+). Wenn ihr auf ein Wort klickt, könnt ihr das Wort ändern; Wenn ihr auf ein + klickt, könnt ihr ein neues Wort dort einfügen. Euer neuer Input wird nach dem Titel angezeigt, klickt also auf das + ganz rechts.
    Bei weiterer Interesse könnt ihr euch die Snap! Anleitung auf Seite 19-20 angucken (engl) und für fortgeschrittene die Seiten 27-31.

  • Gebt dem Input einen Namen: Wählt ein Inputnamen, wie Größe oder Seitenlänge, der die Funktion dieses Blockes klar macht. Tippt diesen Namen in das Input-Feld.

    Ihr habt zwei Möglichkeiten wie ihr mit den neuen Text umgeht:

    1. Input Name ist automatisch für euch ausgewählt. Das ist was ihr jetzt wollt, also lasst es einfach ausgewählt.
    2. Aber falls ihr den Blocknamen ändern wollt, könnt ihr auf Beschriftung (in diesem Bild grau) klicken und die neuen Wörter eingeben. (Zum Beispiel, um den Blocknamn von zeichne Quadrat zu zeichne Quadrat Bild zu ändern, klickt auf "Beschriftung" und tippe das neue Word "Bild".)

    Eure Blockbeschriftung (Header) hat sich geändert: Dort ist jetzt rechts von Titel eine Variable, die Größe (oder was auch immer ihr ausgewählt habt) heißt:

    Die Variable enthält den Wert, den der Benutzer ihres Blockes in das Eingabefeld setzt. Wenn also ein Benutzer sagt zeichne Quadrat (87) ist der Wert der Variable size 87.

  • Um die Variable size zu benutzen, ziehe die Variable dort hin wo ihr sie benötigt. In diesem Falls wird sie im gehe-Block benötigt. Wenn ihr an einer anderen Stelle eine Kopie dieser Variable benötigt, zieht sie einfach dort hin.

    Falls ihr eine überflüssige Kopie eines Blocks oder eines Skripts habt, könnt ihr sie einfach in das Paletten-Fenster auf der linken Seite des snap Fensters ziehen und dort loslassen.
  1. Überprüft eure Arbeit. Wenn ihr im Block-Editor auf "Ok" klickt, werdet ihr sehen, dass der zeichne Quadrat Block im Skript-bereich jetzt ein Input-Feld hat. Führe das Skript mit verschiedenen Inputs aus, um zu überprüfen ob es funktioniert.
  2. Bearbeitet eure Blöcke zeichne Dreieck, zeichne Haus, und zeichne Reihe von Häusern, sodass ihr Dreiecke, Häuser und Reihen von Häusern in jeder Größe zeichnen könnt.
  1. Erstelle ein zeichne Sechseck Block mit einem Input, der die Größe des Sechsecks bestimmt.
  2. Erstelle ein Honigwabe Block (mit einem Input, der die Größe bestimmt), der euren Sechseck-Block dreimal benutzt, um dieses Bild zu erstellen:
 
  1. Erstelle ein zeichne Vieleck Block, der zwei Inputs bekommt:
    • Eine Anzahl an Seiten (nennt die Variable Seitenanzahl)
    • Eine Länge jeder Seite (nennt die Variable Seitenlänge)

    Testet eure Lösung mit mehreren Vielecken, die eine unterschiedliche Seitenanzahl und/oder Seitenlänge besitzen.

Dieser zeichne Vieleck Block generalisiert das Schema von zeichne Quadrat, zeichne Dreieck, zeichne Sechseck und so weiter. Dies geschieht durch Hinzufügen einer Eingabe, die den Unterschied von einem Block zum anderen ausdrückt, nämlich die Anzahl der Seiten. Hinzufügen eines Inputs und ein Schema zu generalisieren ist ein wichtige Technik später in diesem Kurs.

Das ist ein Beispiel für Abstraktion, eine der wichtigsten Konzepte dieses Kurses und von Informatik im Allgemeinen.
